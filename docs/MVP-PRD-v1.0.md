## 一、产品概述

**产品名称：** AI 军团管理平台（AI Corps Platform）

**MVP 目标：** 4 周内上线可用的核心功能，验证"游戏化管理 AI Agent"的核心假设

**核心用户故事：**
> "作为一个小团队负责人，我希望像玩游戏一样管理我的 AI Agent 军团，实时看到它们在做什么，用自然语言下达任务，而不是写代码或拖拽复杂的流程图。"

## 三、功能详细设计

### 3.1 Agent 创建流程

```
用户点击"创建 Agent" 
    ↓
选择模板（客服/写作/调研/代码/自定义）
    ↓
填写基础信息（名称、描述、头像）
    ↓
配置能力（绑定模型、工具、知识库）
    ↓
设置预算（月度 Token 上限）
    ↓
创建完成 → 进入待命状态
```

**界面元素：**
- 模板卡片（带图标和简介）
- 表单（名称/描述/头像选择器）
- 模型下拉框（显示价格和速度）
- 工具多选（搜索/计算器/代码执行/文件读写）
- 知识库选择（可创建或绑定现有）

---

### 3.2 自然语言任务创建

**输入示例：**
```
"每天早上 9 点，扫描 AI 新闻，生成简报发我飞书"
```

**系统解析：**
```json
{
  "intent": "create_scheduled_task",
  "entities": {
    "schedule": "0 9 * * *",
    "action": "scan_news",
    "topic": "AI",
    "output": "briefing",
    "delivery": {
      "channel": "feishu",
      "target": "user"
    }
  },
  "agent_suggestion": "research_agent"
}
```

**交互流程：**
1. 用户输入自然语言
2. 系统解析并展示确认卡片
3. 用户确认或修改
4. 任务创建成功

### 3.4 任务执行流程

```
任务创建
    ↓
意图识别 → 分发给合适 Agent
    ↓
Agent 领取任务 → 状态变为"忙碌"
    ↓
执行任务（可调用工具/API）
    ↓
记录日志（实时推送至面板）
    ↓
任务完成 → 输出结果
    ↓
状态恢复"空闲" → 效能统计更新
```

**异常处理：**
- 超时（默认 30 分钟）→ 自动取消，告警通知
- 错误（API 失败/解析错误）→ 重试 3 次，仍失败则标记异常
- 预算超支 → 暂停任务，通知用户

### 4.3 数据模型（核心表）

```sql
-- Agent 表
CREATE TABLE agents (
  id UUID PRIMARY KEY,
  name VARCHAR(100) NOT NULL,
  description TEXT,
  avatar_url VARCHAR(500),
  model_id VARCHAR(50),
  status VARCHAR(20) DEFAULT 'idle', -- idle, busy, error, offline
  budget_monthly DECIMAL(10,2),
  created_at TIMESTAMP DEFAULT NOW(),
  user_id UUID REFERENCES users(id)
);

-- 任务表
CREATE TABLE tasks (
  id UUID PRIMARY KEY,
  title VARCHAR(200) NOT NULL,
  description TEXT,
  agent_id UUID REFERENCES agents(id),
  status VARCHAR(20) DEFAULT 'pending', -- pending, running, completed, failed
  priority INTEGER DEFAULT 0,
  scheduled_at TIMESTAMP,
  completed_at TIMESTAMP,
  result JSONB,
  created_at TIMESTAMP DEFAULT NOW()
);

-- 日志表
CREATE TABLE task_logs (
  id UUID PRIMARY KEY,
  task_id UUID REFERENCES tasks(id),
  level VARCHAR(10), -- info, warning, error
  message TEXT,
  metadata JSONB,
  created_at TIMESTAMP DEFAULT NOW()
);

-- 效能统计表
CREATE TABLE agent_metrics (
  agent_id UUID PRIMARY KEY,
  total_tasks INTEGER DEFAULT 0,
  success_rate DECIMAL(5,2),
  avg_duration_seconds DECIMAL(10,2),
  total_tokens INTEGER DEFAULT 0,
  total_cost DECIMAL(10,2),
  last_updated TIMESTAMP DEFAULT NOW()
);
```
